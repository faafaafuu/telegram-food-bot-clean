Admin UI — инструкция

Коротко
- Страница `webapp/admin/index.html` — простая панель управления меню: категории и товары. Она использует существующие backend-эндпойнты `/api/admin/*` и `/api/*` для чтения/записи данных.

Как редактировать меню (админы)
- Откройте в браузере: `https://<ваш-base-url>/webapp/admin/` (в dev с ngrok — `https://.../webapp/admin/`).
- Добавляйте категории и товары через форму вверху (они сохраняются в базе данных).
- Для правки элемента нажмите `Save` рядом с ним — поля редактируются inline.
- Для удаления — `Del`.

Технически: где править фронтенд
- Клиентская SPA (для пользователей) находится в `webapp/index.html` и `webapp/app.jsx`.
- Если вам нужно изменить текст интерфейса, стили или поведение — отредактируйте эти файлы в репозитории и перезапустите сервисы (или пересоберите контейнер).

Как применить изменения (простая инструкция)
1) Внесите правки в файлы `webapp/*` в репозитории на сервере.
2) Пересоберите образ и перезапустите стек:

```bash
cd /root/telegram-food-bot-clean
docker compose -f docker/docker-compose.prod.yml up -d --build
```

Советы
- Для быстрых правок в dev можно редактировать `webapp/index.html` прямо на сервере и перезагрузить nginx (или перезапустить web контейнер).
- Для расширения админки (например, загрузка фото для товара) потребуется добавить поле `image_url` в модель и обновить соответствующие эндпойнты и форму.
